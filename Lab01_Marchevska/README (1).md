# 🧠 Програмування систем на кристалі (PSoC)

## 📌 Опис

Цей репозиторій містить лабораторні роботи з дисципліни "Програмування
систем на кристалі".

------------------------------------------------------------------------

## ⚙️ Встановлення PSoC Creator

1.  Завантажити PSoC Creator 4.4
2.  Встановити IDE
3.  Підключити плату через USB

------------------------------------------------------------------------

## 🚀 Запуск

-   Build (Shift+F6)
-   Program (Ctrl+F5)

------------------------------------------------------------------------

# 🔬 Лабораторна робота №1

## 🎯 Мета

Ознайомлення з PSoC Creator та роботою GPIO.

## ⚙️ Виконання

-   Button → P0.7
-   LED_R → P1.6
-   LED_G → P0.2
-   LED_B → P0.3

## 💻 Код

``` c
#include "project.h"

int main(void)
{
    CyGlobalIntEnable;

    for(;;)
    {
        if(Button_Read() == 1)
        {
            LED_R_Write(0);
            LED_G_Write(1);
            LED_B_Write(0);
        }
        else
        {
            LED_R_Write(0);
            LED_G_Write(0);
            LED_B_Write(1);
        }
    }
}
```

## ✅ Висновок

Отримано навички роботи з GPIO та RGB LED.
