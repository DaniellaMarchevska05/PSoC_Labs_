# Хід виконання роботи

## 1. Створення проєкту

CY8C4245AXI-483\
Тип: Empty schematic

------------------------------------------------------------------------

## 2. Додавання компонентів

-   Digital Input Pin --- Button\
-   Digital Output Pin --- LED_R, LED_G, LED_B

------------------------------------------------------------------------

## 3. Налаштування компонентів

Button: - Resistive Pull Up - Initial State: High (1)

LED: - Strong Drive - Initial State: Low (0)

------------------------------------------------------------------------

## 4. Налаштування пінів

Button --- P0.7\
LED_R --- P1.6\
LED_G --- P0.2\
LED_B --- P0.3

------------------------------------------------------------------------

# Програмна реалізація

``` c
#include "project.h"

int main(void)
{
    CyGlobalIntEnable;

    for(;;)
    {
        if(Button_Read() == 1)
        {
            LED_R_Write(0);
            LED_G_Write(1);
            LED_B_Write(0);
        }
        else
        {
            LED_R_Write(0);
            LED_G_Write(0);
            LED_B_Write(1);
        }
    }
}
```
