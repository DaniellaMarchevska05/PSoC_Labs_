# Лабораторна робота №4

## Використання зсувних регістрів та семисегментних індикаторів у PSoC Creator

---

## Мета роботи
Ознайомитися з принципами роботи зсувних регістрів (74HC595) та семисегментних індикаторів,  
навчитися виводити дані та керувати відображенням символів.

---

## Що реалізовано

- Робота із зсувним регістром 74HC595
- Виведення цифр на 7-сегментний індикатор
- Відображення натиснутої кнопки з клавіатури
- Підтримка символів: A, b, C, d, E, F
- Інтеграція з матричною клавіатурою
- Вивід через UART

---

## Принцип роботи

Зсувний регістр дозволяє передавати дані послідовно (по одному біту),  
але отримувати їх паралельно на виходах (Q0–Q7).

- DS — передача біта  
- CLK — зсув  
- Latch — оновлення виходів  

---

## Масив символів

```c
static uint8_t LED_NUM[] = {
    0xC0, 0xF9, 0xA4, 0xB0,
    0x99, 0x92, 0x82, 0xF8,
    0x80, 0x90, 0x88, 0x83,
    0xC6, 0xA1, 0x86, 0x8E
};
```

---

## Відправка даних

```c
static void sendData(uint8_t data)
{
    for(uint8_t i = 0; i < 8; i++)
    {
        if(data & (0x80 >> i))
            Pin_DO_Write(1);
        else
            Pin_DO_Write(0);

        Pin_CLK_Write(1);
        Pin_CLK_Write(0);
    }
}
```

---

## Вивід на індикатор

```c
static void sendDigit(uint8_t position, uint8_t digit)
{
    sendData(0xFF & ~(1 << position));
    sendData(LED_NUM[digit]);

    Pin_Latch_Write(1);
    Pin_Latch_Write(0);
}
```
