# Хід виконання роботи

## 1. Модифікація проєкту

Було використано проєкт з Лабораторної роботи №2 та додано:
- матричну клавіатуру (4 рядки, 3 стовпці)
- bidirectional піни для колонок
- input піни для рядків

------------------------------------------------------------------------

## 2. Налаштування компонентів

Columns:
- Type: Bidirectional  
- Drive Mode: Open Drain (drives low)  
- Initial State: High  

Rows:
- Type: Digital Input  
- Drive Mode: High Impedance  

------------------------------------------------------------------------

## 3. Алгоритм роботи

- послідовно активується кожен стовпець  
- зчитуються рядки  
- визначається натиснута кнопка  
- значення передається в UART  

------------------------------------------------------------------------

# Програмна реалізація

```c
#include "project.h"

static uint8_t keys[4][3];

void initMatrix()
{
    // ініціалізація матриці
}

void readMatrix()
{
    // сканування клавіатури
}

int main(void)
{
    CyGlobalIntEnable;

    SW_Tx_UART_Start();
    initMatrix();

    for(;;)
    {
        readMatrix();

        if(keys[0][0] == 0)
        {
            SW_Tx_UART_PutString("Button 1 pressed");
            SW_Tx_UART_PutCRLF();
        }
    }
}
```

------------------------------------------------------------------------

# Інструкція запуску

1. Відкрити PSoC Creator  
2. Відкрити проєкт  
3. Build → Build Project  
4. Debug → Program  
5. Відкрити PuTTY або TeraTerm  
6. Натискати кнопки клавіатури
