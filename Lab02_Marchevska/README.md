# Хід виконання роботи

## 1. Модифікація проєкту

Було використано проєкт з Лабораторної роботи №1 та додано:
- RGB світлодіод (LED_R, LED_G, LED_B)
- Компонент Software Transmit UART

------------------------------------------------------------------------

## 2. Налаштування компонентів

LED:
- Type: Digital Output  
- Drive Mode: Strong Drive  
- Initial State: Low (вимкнено)

UART:
- Software Transmit UART
- Використовується для передачі текстових повідомлень у термінал

------------------------------------------------------------------------

## 3. Налаштування пінів

- LED_R --- P1.6  
- LED_G --- P0.2  
- LED_B --- P0.3  

------------------------------------------------------------------------

# Програмна реалізація

```c
#include "project.h"

int main(void)
{
    CyGlobalIntEnable;

    SW_Tx_UART_Start();
    SW_Tx_UART_PutCRLF();
    SW_Tx_UART_PutString("Daniella Marchevska");
    SW_Tx_UART_PutCRLF();

    uint8_t last_state = 0;

    for(;;)
    {
        if(Button_Read() == 1 && last_state == 0)
        {
            last_state = 1;
            SW_Tx_UART_PutString("Button released");
            SW_Tx_UART_PutCRLF();

            LED_R_Write(0);
            LED_G_Write(0);
            LED_B_Write(0);
        }
        else if(Button_Read() == 0 && last_state == 1)
        {
            last_state = 0;
            SW_Tx_UART_PutString("Button pressed");
            SW_Tx_UART_PutCRLF();

            LED_R_Write(1);
            LED_G_Write(1);
            LED_B_Write(1);
        }
    }
}
```
